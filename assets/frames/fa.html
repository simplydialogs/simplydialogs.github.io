Example: Use Font Awesome-icons instead of the default unicode symbols :

Dlg.DEFAULTS.icons = {
  alert: '<i class="fa fa-warning" color="navy"></i>',
  information: '<i class="fa fa-info-circle"></i>',
  confirm: '<i class="fa fa-question-circle-o"></i>',
  bell: '<i class="fa fa-bell-o"></i>',
  error: '<i class="fa fa-times-circle-o"></i>',
  wait: '<i class="fa fa-gear">build</i>',
  input: '<i class="fa fa-pencil">info</i>'
}
